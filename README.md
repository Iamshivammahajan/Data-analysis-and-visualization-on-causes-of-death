# 📊 Data Analysis and Visualization on Causes of Death

A Python-based data analysis project that explores global causes of death using real-world datasets. This application enables users to analyze mortality data, generate insightful visualizations, and export data to a MySQL database through an interactive menu-driven interface.

---

## 📌 Overview

The project demonstrates the use of Python libraries for data analysis and visualization. Users can view the dataset, perform statistical analysis, visualize trends using different graph types, and export processed data to a MySQL database.

---

## ✨ Features

- 📂 Load and process CSV datasets
- 📊 Display complete dataset and column information
- 📈 Analyze minimum and maximum death statistics
- 🌍 Country-wise data analysis
- 📅 Year-wise data analysis
- 📉 Generate multiple visualizations
- 🗄️ Export dataset to MySQL database
- 🖥️ Interactive menu-driven interface

---

## 📊 Visualizations

The project provides the following visualizations:

- 📊 Bar Graph
- 📈 Scatter Plot
- 📉 Histogram
- 🥧 Pie Chart
- 📊 Horizontal Bar Graph

Graphs are generated for various causes of death including:

- Meningitis
- Alzheimer's Disease
- Cardiovascular Diseases
- Lower Respiratory Infections
- Neoplasms

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Data-Analysis-Causes-of-Death.git

cd Data-Analysis-Causes-of-Death
```

### 2. Install Required Libraries

```bash
pip install numpy pandas matplotlib sqlalchemy pymysql
```

or

```bash
pip install -r requirements.txt
```
